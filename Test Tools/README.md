# 测试工具说明

本文件夹包含用于测试Send to AI.ahk脚本的各种测试文件。

## 文件说明

1. **Encoding Test.ahk**
   - 用于测试UTF-8编码读取配置文件
   - 比较UTF-8编码读取和传统IniRead方法的结果
   - 验证中文字符的正确读取

2. **Simple INI Test.ahk**
   - 简单的INI文件读取测试
   - 测试基本的配置文件读取功能

3. **Simple Test.ahk**
   - 基本功能测试脚本
   - 用于测试脚本的基本功能

4. **Test Config.ahk**
   - 配置加载测试脚本
   - 测试配置文件的加载和切换功能

5. **Debug Config.ahk**
   - 调试版本的配置脚本
   - 包含更多调试信息输出

6. **Config_UTF8.ini**
   - UTF-8 BOM版本的配置文件
   - 用于测试UTF-8编码的配置读取

## 使用方法

1. 要运行这些测试，请使用AutoHotkey执行相应的.ahk文件
2. 测试结果会通过消息框显示
3. 某些测试可能需要修改配置文件路径以适应您的环境

## 注意事项

- 这些测试文件仅用于开发和调试目的
- 请勿在生产环境中使用这些测试文件
- 测试完成后，可以将这些文件保留在此处以备将来使用
